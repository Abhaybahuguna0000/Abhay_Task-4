# CSS Units, Box Model & Fonts Task

## Overview
This project demonstrates the use of CSS viewport units, the CSS box model, and font styling. It displays an image with a customized button placed below it.

## Features
- Responsive layout using viewport units (`vw` and `vh`)
- Image and button with equal width
- Gradient button background
- Custom font styling
- Proper spacing using CSS margins
- Box sizing implemented
- No Flexbox or Position properties used
- No page scrolling

## Files
- index.html
- style.css

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download or clone the project.
2. Keep `index.html`, `style.css`, and the image in the same directory.
3. Open `index.html` in any modern web browser.

## Learning Outcomes
- Understanding CSS viewport units (`vw`, `vh`)
- Working with the CSS Box Model
- Applying gradient backgrounds
- Using custom fonts and button styling
- Creating responsive layouts without Flexbox
